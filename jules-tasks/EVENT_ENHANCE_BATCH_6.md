# Event Enhancement Batch 6: Family Attractions & Summer Events

Enhance these 6 Welsh family attraction events with accurate, verified data. **No AI images.**

## Events to Enhance

1. **greenwood-halloween** - https://www.greenwoodfamilypark.co.uk/
2. **folly-farm-halloween** - https://www.folly-farm.co.uk/
3. **zip-world-fright-nights** - https://www.zipworld.co.uk/
4. **greenwood-summer-events** - https://www.greenwoodfamilypark.co.uk/
5. **folly-farm-summer** - https://www.folly-farm.co.uk/
6. **dan-yr-ogof-events** - https://www.showcaves.co.uk/

## For Each Event, Capture:

### 1. Coordinates (REQUIRED)
- Exact attraction coordinates
- These are fixed venues

### 2. Hero Image (REQUIRED)
- Official attraction imagery
- **NO AI-generated images**
- Kid-friendly, action shots

### 3. Social Links
- All these attractions have active Facebook/Instagram
- Get main attraction accounts

### 4. Category
Use: `family`

### 5. Season/Dates
- Halloween: Late Oct
- Summer: School holidays
- Note if it's a season-long offering vs specific dates

### 6. Link to Operator
- Check if these attractions are in the AW operator directory
- Add operator slug for cross-linking

### 7. Pricing
- Typical admission prices
- Whether event is included with regular admission

## Output Format

Save each to `data/events/enhanced/{slug}.json`
